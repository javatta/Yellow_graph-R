# Семантика «желтого» во фразеологии: сопоставительный анализ (Китайский vs Русский)

Этот проект представляет собой визуализацию данных лингвистического исследования, посвященного значениям лексемы «желтый» в устойчивых выражениях и идиомах китайского и русского языков.

## О задании
Данная визуализация была создана в рамках учебного задания.
**Цель задания:** создать «амеренно плохой график. 

## О данных
Данные для визуализации взяты из моего исследования семантических полей желтого цвета во фразеологизмах в русском и китайском языках.
